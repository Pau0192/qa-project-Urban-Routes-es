                      AUTOMATIZACION DE PRUEBAS DE LA APLICACION WEB                                  Proyecto 8

Nos encargamos de realizar en total de 8 pruebas automatizadas, descubriendo los elementos en la aplicación web.
Y así probar las funcionalidades de estos elementos para el correcto uso del usuario. Esta enfocado en las campos 'from y to', 'boton comfort', 'agregar telefono', 'boton método de pago', 'agregar tarjeta', 'agregar mensaje', 'pedir sabana y pañuelos', 'agregar helado' y 'boton pedir taxi'   

Utilizamos principalmente Pycharm, Selenium webdriver, chrome webdriver, pytest y Python3

Instalación

1. Instalar Pycharm para la ejecución de pruebas. Se recomienda PyCharm Community
2. Descargar Chrome webdirver y configurar la ruta en tu sistema (PATH). 
3. Instalar Selenium (compatible con tu chrome), utiliza el comando pip Selenium en tu terminal o desde el area de Paquetes lo puedes encontrar
4. Instalar Python en caso de no tenerlo.
5. Instalar pytest en caso de no tenerlo. utiliza el comando pip pytest en tu terminal o desde el area de Paquetes lo puedes encontrar 
6. Clona este repositorio a tu directorio local 
     git@github.com:Pau0192/qa-project-Urban-Routes-es.git

Ejecución de pruebas

Para realizar los test, se realiza con el comando pytest test_urban_routes.py desde la terminal.
